# 🔥 Dark Souls — Who Am I? (DLE Style)

Um jogo de adivinhação diária ambientado no universo de Dark Souls e Bloodborne. Inspirado no estilo "DLE" (como Wordle, Loldle ou Pokedle), o projeto desafia o conhecimento dos jogadores sobre personagens, itens, cenários e trilhas sonoras da franquia.

## 🎮 O Jogo

O objetivo principal é descobrir o **Personagem do Dia**. Além da adivinhação principal, o projeto inclui um sistema de quiz com múltiplas categorias para testar seu conhecimento de Lordran, Yharnam e além.

### Categorias de Adivinhação:
*   **⚔️ Personagens:** Adivinhe o habitante através de atributos como Tipo, Jogo, Altura, Peso, Arma, Raça e Sexo.
*   **🏺 Itens:** Identifique equipamentos, anéis e consumíveis icônicos.
*   **🏰 Cenários:** Reconheça as áreas e locais explorados nos jogos.
*   **🎵 Música:** Teste seu ouvido com as trilhas sonoras e compositores da série.

## ✨ Funcionalidades

*   **Busca Inteligente:** Autocomplete para facilitar a escolha de personagens.
*   **Feedback Visual:** Sistema de cores (verde, amarelo, vermelho) e setas indicativas (↑, ↓) para guiar suas tentativas baseadas na proximidade dos atributos.
*   **Persistência Diária:** O progresso do dia é salvo automaticamente no navegador (`localStorage`).
*   **Compartilhamento:** Gere um resumo da sua partida em emojis para compartilhar com outros *Undeads*.
*   **Estética Soulsborne:** Interface inspirada nos menus dos jogos, com efeitos de partículas de brasas e fontes clássicas.

## 🛠️ Tecnologias Utilizadas

*   **HTML5 & CSS3:** Estrutura e estilização premium com foco em experiência visual.
*   **JavaScript (Vanilla):** Lógica do jogo, sistemas de comparação e manipulação de estado.
*   **Canvas API:** Utilizada para o sistema de partículas de fundo.
*   **Google Fonts:** Tipografia temática (*Cinzel* e *Crimson Text*).

## 🚀 Como Executar

Como o projeto é construído totalmente com tecnologias web padrão (sem frameworks), basta abrir o arquivo `index.html` em qualquer navegador moderno.
